# realsense-jetson-nano

For a nicer development experience, the code can also be built and run on MacOS.

## Building and running

1. Install [RealSense SDK](https://github.com/IntelRealSense/librealsense)
   - MacOS: `brew install librealsense`
   - Jetson Nano/Ubuntu (JetPack 4.3): Instructions [here](https://github.com/IntelRealSense/librealsense/blob/master/doc/installation_jetson.md)
2. Build with `make`
3. Run with `./main`
