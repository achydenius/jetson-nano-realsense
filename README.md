# realsense-jetson-nano

For a nicer development experience, the code can also be built and run on MacOS.

## MacOS installation

1. Install [RealSense SDK](https://github.com/IntelRealSense/librealsense) with `brew install realsense`
2. Install [Raylib](https://www.raylib.com) with `brew install raylib`
3. Build with `make`
4. Run with `./main`
